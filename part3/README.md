# Part 3 - Persistence, ORM, and Database Design

This part introduces SQLAlchemy-based persistence and relational modeling for HBnB.

## What Is Included

- SQLAlchemy repository implementation (`hbnb/app/persistence/repository.py`)
- ORM mappings for `User`, `Place`, `Review`, and `Amenity`
- Relationship mappings (one-to-many and many-to-many)
- SQL schema and seed scripts (`hbnb/sql/`)
- Mermaid ER diagram (`hbnb/docs/er_diagram.md`)

See `hbnb/README.md` for full setup, architecture, and usage details.
